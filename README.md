# Лабораторна робота №2: Обробка XML документів

Програма для аналізу, редагування та трансформації XML-документів на тему "Розклад занять" (TimeTable). Реалізовано з використанням **.NET MAUI**.

##  Інформація про студента

* **Виконала:** Соболева Софія
* **Група:**  К-24
* **Варіант:** 5 ("Розклад")

---

##  Опис предметної області 

Програма працює з XML-файлами, що містять структуру **Розкладу занять**.
Основні сутності:
* **Person** (Викладач): Ім'я, Прізвище, Факультет, Кафедра.
* **Class** (Заняття): Предмет, Дата (День, Час), **Аудиторія**.
* **Students** (Студенти): Список студентів, присутніх на парі (Ім'я, Група).

##  Функціональні можливості

1.  **Відкриття та валідація:** Завантаження XML-файлів з перевіркою відповідності XSD-схемі.
2.  **Парсинг (Стратегії):** Реалізовано три підходи до зчитування файлу:
    * **DOM** (XmlDocument)
    * **SAX** (XmlReader)
    * **LINQ to XML** (XDocument)
3.  **Пошук та фільтрація:** Можливість фільтрувати заняття за різними критеріями (наприклад, за факультетом, аудиторією, викладачем тощо).
4.  **Трансформація:** Експорт відфільтрованих даних у формат **HTML** для зручного перегляду (використовується XSLT).
5.  **Очищення:** Можливість скинути фільтри та очистити результати.

---

##  Структура проекту

* **`GUI`** — Графічний інтерфейс користувача (MAUI).
* **`XML_Utils`** — Бібліотека класів з логікою парсерів (DOM, SAX, LINQ) та моделями даних.
* **`Resources/Raw`** — Містить файли `schema.xsd` (валідація) та `schema.xsl` (трансформація).
###  Тестові дані

Приклад готового файлу для перевірки роботи програми знаходиться у папці проекту:
 **`src/GUI/Data/TimeTable.xml`**

Ви можете використати цей файл, щоб одразу побачити результати роботи парсерів.
